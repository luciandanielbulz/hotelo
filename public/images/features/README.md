# Feature-Bilder

Legen Sie hier die Bilder für die Feature-Karten auf der Startseite ab.

## Dateinamen:

1. **rechnungen.jpg** oder **rechnungen.png** - Für "Rechnungen verwalten"
2. **angebote.jpg** oder **angebote.png** - Für "Angebote erstellen"
3. **kundenverwaltung.jpg** oder **kundenverwaltung.png** - Für "Kundenverwaltung"
4. **pdfs.jpg** oder **pdfs.png** - Für "Professionelle PDFs"
5. **versand.jpg** oder **versand.png** - Für "Direkter Versand"
6. **analysen.jpg** oder **analysen.png** - Für "Übersicht & Analyse"

## Empfohlene Bildgröße:
- **Breite:** 800-1200px
- **Höhe:** 600-800px
- **Format:** JPG oder PNG
- **Optimierung:** Bilder sollten für Web optimiert sein (komprimiert)

## Alternative Dateinamen:
Falls Sie andere Namen verwenden möchten, können Sie diese in der `welcome.blade.php` anpassen.
